Preliminary Results and Accomplishments

To start off our project, we decided that it would be the easiest for us to approach it by modifying the Project 4 code, rather than building an entire framework/system from scratch. In our modifications to the code, we are in the process of changing the cloth grid system to represent a fluid instead. To do this, we have begun making changes such as replacing the pointMasses making up cloth to particles, which are very similar in concept but have additional attributes, such as velocity, density, etc. We have also been working on modifying the construction of the cloth, so that instead of creating a 2D grid, we can create a 3D cube of fluid that is made up of these particles. However, these modifications have given us a lot of difficulty, since making many of these modifications involve having a good understanding of much of the staff-provided code. We are still struggling to get decent outputs after making much of these changes. We have also started looking into the ways in which we would have to modify the simulate() function, since the way in which we update positions for fluid is quite different than how it is done for cloth. This has involved looking into constraints such as incompressibility and viscosity.

We also played around with rendering tools and learned how to render a frame using a water BSDF using Mitsuba. We did so by creating an xml file that describes the scene, along with a water BSDF, and used those with a pre-simulated obj file to create this image. This pre-simulated obj file is what our simulator generates for us, which we then properly render using the aforementioned tools and methods.
